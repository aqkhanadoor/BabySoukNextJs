## Project Information

This project is an e-commerce website for baby products.

### Technologies Used
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui

### How to run locally

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Start the development server: `npm run dev`
